## Running Weverse Shop

### Clone project
```bash
git clone https://github.com/MELLI-uni/weverse-shop.git
cd weverse-shop
```

### Install yarn dependecies
```bash
yarn add
```

### Run Weverse Shop
```bash
yarn dev
```

This will start up the local Weverse Shop app on http://localhost:3000.
